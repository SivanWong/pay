## 使用
1. 从github中下载
2. npm install安装依赖
3. npm run dev运行
4. 入口文件index有两个不同车牌的入口
5. 点击 相应车牌 跳转到 对应的缴费页面

## 已完成功能
- 根据ui转换为静态页面
- 根据id模拟请求，向本地json数据请求相应的数据，并展现出来
- 适配移动端
- 利用vue实现数据双向绑定
- 选择支付宝支付提示“无法支付”
- 选择微信支付提示“支付成功”
- 根据渲染前请求得到的数据中的未缴费或已缴费来判断并决定选择列表与支付按钮是否显示
